## 🖼️ 应用功能简介

Image Converter 最初是为了解决一个非常实际的问题而创建的：参考图片文件夹的整理。  
它帮助艺术创作者保持图片参考库的整洁，并节省手动重命名所花费的时间。

如果你经常从 Pinterest（尤其是）、ArtStation、Behance 或 Google Images 收集大量参考图片，你一定遇到过这些问题：文件名混乱、扩展名异常，以及仅仅“重命名为 .jpg”并不能保证图片真正可用。

因此，这款应用在一个清晰的工作流程中同时完成两件事：  
**批量重命名**（使用可预测的命名规则）和 **真实格式转换**，确保图片文件能够稳定、可靠地打开。
&nbsp;
&nbsp;

## 🎨 功能特性

- 将杂乱的图片文件批量重命名为统一格式，如 `ref_001.jpg`、`ref_002.jpg`……
- 转换为标准格式（JPG/JPEG、PNG），确保文件可正常打开。
- 自动处理重复文件名，添加 `_copy`、`_copy2` 等后缀。
- 清理文件名中的非法字符。
- 显示处理进度，跳过损坏的文件，保持流程简单直观。
- 简洁、极简风格的用户界面。

## ⚒ 安装方式

- 前往 **Releases** 页面，下载最新的 `.exe` 文件。
- 解压到任意位置（桌面、工具文件夹、U 盘等）。
- 运行 `Image Converter.exe`。
&nbsp;
&nbsp;

## 🏓 使用方法

1. 选择输入文件夹。
2. 选择输出格式。
3. 设置文件名规则（例如：`ref_###` → `ref_001`、`ref_002`……）。
4. 点击 **Start**，等待处理完成。
